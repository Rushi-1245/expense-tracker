# Expense Tracker

A modern Expense Tracker application built with **SwiftUI** that helps users manage personal finances by tracking income and expenses, organizing transactions, and visualizing financial insights through interactive charts.

## ✨ Features

### 💰 Income Management
- Add income records
- Categorize income sources
  - Salary
  - Business
  - Interest
  - Rental
  - Other

### 💸 Expense Management
- Add expense records
- Categorize expenses
  - Food
  - Shopping
  - Entertainment
  - Travel
  - Health
  - Other
- Add notes to expenses

### 📊 Dashboard
- Financial overview at a glance
- Track income and expenses
- Quick access to transaction management
- Clean and intuitive user interface

### 📋 Transactions
- Complete transaction history
- Filter transactions by date
- Sort transactions by:
  - Date
  - Amount
  - Name
- Ascending and descending ordering

### 📈 Insights & Analytics
- Monthly financial analysis
- Expense distribution charts
- Income source breakdown
- Interactive donut charts using Swift Charts

### 🎨 User Experience
- Modern SwiftUI design
- Responsive layouts
- Tab-based navigation
- Smooth user interactions

## 🏗 Architecture

This project follows the **MVVM (Model-View-ViewModel)** architecture pattern.

### Architectural Highlights

- MVVM Architecture
- Protocol-Oriented Programming
- Dependency Injection
- Generic Database Layer
- Reusable SwiftUI Components

- Scalable Project Structure

## 🚀 Getting Started

- Xcode 16+
- iOS 18+
- Swift 6

### Clone Repository

